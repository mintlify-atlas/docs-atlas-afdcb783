# Welcome to Teak

Teak is a personal knowledge hub designed to help creative minds effortlessly collect, remember, and rediscover their most important ideas and inspirations.

Whether you're saving a brilliant article, capturing a fleeting thought, or archiving visual inspiration, Teak helps you build a searchable, organized repository of everything that matters to you.

## Card Types

Teak supports 8 different card types to capture any kind of information:

- **Text** - Quick notes, thoughts, and ideas
- **Link** - Web pages with automatic metadata extraction
- **Image** - Visual inspiration and screenshots
- **Video** - Video content with AI transcription
- **Audio** - Voice notes and audio clips with transcripts
- **Document** - PDFs and documents
- **Palette** - Color palettes for design inspiration
- **Quote** - Memorable quotes and passages

## Platforms

Teak is available across all your devices:

- **Web App** - Access at [teakvault.com](https://teakvault.com)
- **Mobile** - Native iOS app via App Store
- **Desktop** - Native desktop application for macOS, Windows, and Linux
- **Browser Extension** - Quick-save from any webpage
- **Raycast** - Fast access via Raycast command palette

## AI Processing

Every card you create is automatically enhanced with AI:

1. **Classification** - Automatically detect card type and extract colors from images
2. **Metadata Extraction** - Pull titles, descriptions, and thumbnails from links
3. **Smart Tagging** - AI-generated tags for easy organization
4. **Summarization** - Automatic summaries of long content
5. **Transcription** - Speech-to-text for audio and video content
6. **Thumbnails** - Generated previews for media content
