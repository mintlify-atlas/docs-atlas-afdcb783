# Installation

Teak is available across multiple platforms. Install it on all your devices to capture ideas wherever you are.

## Web App

The easiest way to get started is through the web app. No installation required.

<Steps>
  <Step title="Visit Teak">
    Navigate to [teakvault.com](https://teakvault.com) in any modern web browser.
  </Step>
  
  <Step title="Sign Up or Sign In">
    Create an account or sign in with your existing credentials.
  </Step>
  
  <Step title="Start Creating Cards">
    You're ready to go! Start saving your first cards.
  </Step>
</Steps>

<Note>
  The web app works on all modern browsers including Chrome, Firefox, Safari, and Edge. For the best experience, we recommend using the latest version of your browser.
</Note>

## Mobile App (iOS)

Capture ideas on the go with the native iOS mobile app built with Expo.

<Steps>
  <Step title="Download from App Store">
    Search for "Teak" in the iOS App Store or visit the direct App Store link.
  </Step>
  
  <Step title="Install the App">
    Tap "Get" to download and install Teak on your iPhone or iPad.
  </Step>
  
  <Step title="Sign In">
    Open the app and sign in with the same account you created on the web.
  </Step>
  
  <Step title="Grant Permissions (Optional)">
    Allow camera and photo library access to quickly capture images and upload media to your knowledge hub.
  </Step>
</Steps>

<Note>
  Your cards sync in real-time across all devices via Convex, so anything you save on mobile appears instantly on web and desktop.
</Note>

## Desktop App

Get a native desktop experience with the Tauri-based desktop application.

<Tabs>
  <Tab title="macOS">
    <Steps>
      <Step title="Download">
        Download the `.dmg` installer for macOS from the Teak website.
      </Step>
      
      <Step title="Install">
        Open the `.dmg` file and drag Teak to your Applications folder.
      </Step>
      
      <Step title="Launch">
        Open Teak from your Applications folder and sign in.
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Windows">
    <Steps>
      <Step title="Download">
        Download the `.exe` installer for Windows from the Teak website.
      </Step>
      
      <Step title="Install">
        Run the installer and follow the setup wizard.
      </Step>
      
      <Step title="Launch">
        Open Teak from the Start Menu and sign in.
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Linux">
    <Steps>
      <Step title="Download">
        Download the `.AppImage` or `.deb` package from the Teak website.
      </Step>
      
      <Step title="Install">
        For AppImage: Make it executable (`chmod +x Teak.AppImage`) and run it.
        For .deb: Install with `sudo dpkg -i teak.deb`.
      </Step>
      
      <Step title="Launch">
        Open Teak and sign in.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Browser Extension

Save links, images, and content directly from your browser with one click.

<Steps>
  <Step title="Install from Chrome Web Store">
    Visit the Chrome Web Store and search for "Teak" or use the direct link to the extension page.
    
    Click "Add to Chrome" to install the extension.
    
    <Note>
      The extension is built with Wxt and works on all Chromium-based browsers including Chrome, Edge, Brave, and Arc.
    </Note>
  </Step>
  
  <Step title="Pin the Extension">
    Click the puzzle icon in your browser toolbar and pin Teak for easy access.
  </Step>
  
  <Step title="Sign In">
    Click the Teak icon and sign in with your account.
  </Step>
  
  <Step title="Save Your First Link">
    Navigate to any webpage and click the Teak icon to save it as a card. The extension will automatically extract the title, description, and thumbnail.
    
    You can also right-click on any link or image and select "Save to Teak" from the context menu.
  </Step>
</Steps>

### Extension Features

- **Auto-save links**: Automatically save the current page with one click
- **Context menu**: Right-click any link or image to save it
- **Quick popup**: View recent cards and search without leaving your browser
- **Real-time sync**: Cards saved via extension appear instantly on all devices

## Raycast Extension

Access your knowledge hub at lightning speed with the Raycast extension.

<Steps>
  <Step title="Install Raycast">
    If you don't have Raycast installed, download it from [raycast.com](https://raycast.com).
  </Step>
  
  <Step title="Install Teak Extension">
    Open Raycast and search for "Store" to open the Extension Store.
    
    Search for "Teak" and click Install.
  </Step>
  
  <Step title="Configure API Access">
    After installation, you'll need to configure the extension with your Teak API credentials.
    
    <Steps>
      <Step title="Get Your API Key">
        Go to [teakvault.com/settings](https://teakvault.com/settings) and navigate to the API section.
        
        Generate a new API key if you don't have one.
      </Step>
      
      <Step title="Enter API Key in Raycast">
        Open Raycast preferences > Extensions > Teak.
        
        Paste your API key in the configuration field.
      </Step>
    </Steps>
  </Step>
  
  <Step title="Start Using Teak">
    Open Raycast (Cmd + Space) and type "Teak" to see available commands:
    
    - **Search Cards** - Search your entire knowledge hub
    - **Create Card** - Quickly create a new card
    - **Recent Cards** - View your most recent cards
    - **Favorites** - Access your starred cards
  </Step>
</Steps>

<Note>
  The Raycast extension provides the fastest way to search and create cards without leaving your keyboard.
</Note>

## Troubleshooting

### Web App Issues

- **Can't sign in**: Clear your browser cache and cookies, then try again
- **Cards not loading**: Check your internet connection and refresh the page

### Mobile App Issues

- **Sync not working**: Make sure you're signed in with the same account on all devices
- **Upload failing**: Check your internet connection and storage permissions

### Desktop App Issues

- **App won't launch**: Make sure you have the latest version installed
- **macOS security warning**: Go to System Preferences > Security & Privacy and allow the app to run

### Browser Extension Issues

- **Extension not saving**: Make sure you're signed in to the extension
- **Context menu not appearing**: Refresh the page after installing the extension

### Raycast Extension Issues

- **API key not working**: Generate a new API key from your Teak settings
- **Search not returning results**: Make sure your API key is correctly configured
